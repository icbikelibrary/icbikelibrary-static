🚲 IC Bike Library — Static Website

A community-driven static website for the IC Bike Library, focused on promoting accessible cycling, sustainability, and community engagement.


About the Project

This repository contains the full static site powering the IC Bike Library’s online presence. It provides information about programs, events, resources, and initiatives that support cycling and community development.

The site is built entirely with static technologies, making it lightweight, fast, and easy to maintain.


Key Features

- 🚴 Information on bike programs and services
- 📅 Events, rides, and community activities
- 🛠️ Repair workshops and educational content
- 🤝 Volunteer and outreach initiatives
- 📱 Mobile-friendly navigation and layout
- 🖼️ Media-rich pages with images and content


🧱 Project Structure
.
├── *.html                 # Static pages (routes, events, programs, etc.)
├── files/                 # Assets (images, favicon, icons)
├── uploads/               # Uploaded media content
├── apps/                  # Additional scripts or integrations
├── theme/                 # Theme-related styles and assets
├── main_style.css         # Main stylesheet
└── favicon.ico            # Site icon (may also be referenced from /files)


⚙️ Technologies

- HTML5
- CSS3
- JavaScript
- jQuery


🚀 Getting Started

Clone the repository:

git clone https://github.com/icbikelibrary/icbikelibrary-static.git
cd icbikelibrary-static

Open the site locally:

index.html

You can open it directly in your browser — no server setup required.


🧩 Implementation Notes

- The site is fully static (no backend dependencies)
- The favicon is referenced from:
files/favicon.ico
- JavaScript is used for UI interactions (menus, cart behavior, navigation)
- Pages follow a consistent structure for easy maintenance
